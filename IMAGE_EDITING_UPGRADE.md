# Image Editing Upgrade

This build adds direct photo/image editing for a more natural mobile-first workflow.

## Added

- Photo blocks can be dragged directly on the canvas.
- Selected blocks now show resize handles on corners and edges.
- Photos support Fill/Fit display modes.
- Photos support focus controls for object positioning.
- Photos support radius and shadow controls.
- Resize handles are hidden during preview and export.

## Mobile workflow

1. Tap **Insert**.
2. Add **Photo**.
3. Choose an image.
4. Drag the photo to reposition it.
5. Drag any handle to resize it.
6. Use **Edit** for Fill/Fit, focus, radius, shadow, opacity, and exact size.
